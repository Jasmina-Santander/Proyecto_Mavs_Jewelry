Mavs Jewelry
Sitio web en vivo: https://mavs-jewelry.netlify.app/
 
Este proyecto es un sitio web responsivo diseñado para un emprendimiento de joyería artesanal fina, especializada en piezas de plata nacional, piedras semipreciosas, mostacillas calibradas Miyuki y técnicas avanzadas de alambrismo. El objetivo principal de esta plataforma es transmitir la elegancia, la resiliencia y el misticismo detrás de cada joya hecha a mano por su fundadora. 
 
La Evolución del Proyecto 
 
Este sitio siguió un proceso de diseño y maduración por etapas: 
 
1. El Mockup Inicial (Diseño): Comenzó con una etapa de conceptualización visual. Se estructuraron las secciones clave (Inicio, Nosotros, Productos, Concursos, Testimonios y Contacto) pensando en los requerimientos que obtuvimos por parte del emprendedor. 

2. Estructura y Estilos Base (CSS): Una vez definida la idea, se maquetó en HTML5 y se construyeron los cimientos visuales con CSS. Se añadieron personalizaciones estéticas para reflejar la identidad de la marca. 
 
3. Migración y Optimización Responsiva: El proyecto se migró e integró con una arquitectura limpia basada en una plantilla optimizada.  
 
4. Interactividad y Validación (JavaScript): Finalmente, se implementó un script de JavaScript nativo (Vanilla JS) personalizado y aislado para el formulario de contacto, el cual, valida rigurosamente los campos de nombre y formato de correo electrónico, ofreciendo respuestas amigables al usuario y previniendo comportamientos inesperados en la página. 
 
Tecnologías Utilizadas 
 
- HTML5 semántico: Para una estructura accesible y bien indexada. 
- CSS3 Avanzado (Flexbox, Grid, Media Queries, Scroll-Snap): Para lograr un diseño completamente responsivo y adaptado a dispositivos móviles. 
- JavaScript nativo (Vanilla JS): Para la lógica de validación del lado del cliente y manejo de eventos del formulario. 
- jQuery y Dropotron: Como soporte para los componentes de navegación y animaciones de la interfaz de la plantilla.
